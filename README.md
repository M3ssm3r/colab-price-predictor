# House Price Prediction Notebook

A Google Colab notebook demonstrating a complete machine learning workflow for predicting house prices using the California Housing dataset. This project covers data exploration, feature importance, model training, evaluation, and visualization of results.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/M3ssm3r/colab-price-predictor/blob/main/House_Price_Prediction.ipynb)

## Workflow Demonstrated

-   **Data Loading & Exploration:** Using Pandas and Scikit-learn.
-   **Data Visualization:** Using Matplotlib and Seaborn to understand feature distributions.
-   **Model Training:** Training a `RandomForestRegressor` model.
-   **Feature Importance:** Analyzing which features contribute most to the prediction.
-   **Evaluation:** Calculating metrics like Mean Absolute Error and RÂ² score.
-   **Visualization of Predictions:** Plotting actual vs. predicted values to visually assess model performance.

## Tech Stack
-   **Cloud Environment:** Google Colab
-   **ML Library:** Scikit-learn
-   **Data Manipulation:** Pandas, NumPy
-   **Visualization:** Matplotlib, Seaborn
